# OpenFlasher Beta

Samsung flashing tool for Linux (Heimdall frontend) with integrated Magisk patch workflow.

## Current status
- Beta release
- Magisk tab is marked as **Magisk Beta**

## Requirements
- Linux x86_64
- `heimdall`, `lz4`, `tar`, `usbutils`
- Python 3 + PyQt6 (for source run)

## Run from source
```bash
python3 main.py
```

## Notes
- Flashing operations require root privileges.
- Always verify firmware compatibility before flashing.
